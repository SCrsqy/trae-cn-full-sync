# 六合彩历史数据统计与预测系统

## 项目简介

基于 Python Flask 构建的六合彩历史数据统计与预测系统，支持澳门和香港两个地区的数据爬取、统计分析和预测生成。

## 功能特性

- **数据爬取**: 自动从指定网址抓取历史开奖数据
- **农历生肖判定**: 根据公历日期自动转换为农历年份生肖
- **统计分析**: 数字频率统计、生肖频率统计、三肖组合共现分析
- **预测生成**: 特码6只、4组三肖、热门12数字预测
- **准确率计算**: 自动回填并计算命中率
- **投资收益分析**: 模拟投资收益计算
- **Excel导出**: 支持单地区和综合数据导出

## 技术栈

- Python 3.8+
- Flask
- requests + BeautifulSoup (数据爬取)
- pandas + openpyxl (Excel导出)
- SQLite (数据库)

## 快速开始

### macOS/Linux

```bash
cd lhc_prediction
chmod +x start.sh
./start.sh
```

### Windows

```cmd
cd lhc_prediction
start.bat
```

### 手动启动

```bash
pip install -r requirements.txt
python app.py
```

## 访问地址

启动后访问: http://localhost:5000

## 使用说明

1. **爬取数据**: 点击"爬取数据"按钮从远程服务器获取历史开奖数据
2. **生成预测**: 输入期号，点击"生成预测"按钮
3. **录入特码**: 输入期号和实际特码，系统自动计算准确率
4. **导出Excel**: 支持单地区导出和综合导出
